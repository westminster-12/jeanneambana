Project information

I developed a skip counting learning application designed for elementary school students while working at the Ministry of Education, in collaboration with the Learning Technology Development Team. The application features unique animations and simplified explanations of skip counting concepts, making it easy for young learners to understand. Additionally, it includes interactive features for answering questions, enhancing the learning experience.

Key Features:

  • Engaging Animations: Unique animations to make learning skip counting fun and engaging.
  • Child-Friendly Explanations: Simplified explanations of skip counting tailored for elementary school students.
  • Interactive Questions:: Features for answering questions to reinforce learning.
  • Cross-Platform Support: Available on both Android and Windows platforms for broad accessibility.

Technologies Used

  • C#, Unity
