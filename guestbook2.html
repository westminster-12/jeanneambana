Guest Book (v.2)

Project information

This project represents an updated version of the Buku Tamu web application tailored for the Ministry of Education. The new version features a mobile-friendly interface, enhanced security measures, and robust admin capabilities. Users can now effortlessly update and delete guest data, benefiting from an improved, visually appealing, and interactive display.

Key Features:

  • Mobile-Friendly Interface: Optimized for use across various devices within the same local network.
  • Enhanced Security: Strengthened security protocols for safe data handling.
  • Admin Management:: Admin privileges for managing guest entries efficiently.
  • Interactive Display: Improved user interface for better usability and engagement.

Technologies Used

  • HTML, CSS, PHP, JavaScript, MySQL, CodeIgniter 4, Bootstrap
